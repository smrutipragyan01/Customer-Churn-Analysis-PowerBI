# 📊 Customer Churn Analysis Dashboard (Power BI)

## 📌 Project Overview
This project analyzes customer churn in a telecom company to identify key factors affecting customer retention.

## 🛠 Tools Used
- Power BI
- Excel / CSV Dataset

## 📊 Dashboard Features
- Churn Rate Overview
- Customer Demographics (Gender, Partner, Dependents)
- Tenure Distribution
- Churn Analysis (Contract, Payment Method, Internet Service)

## 📈 Key Insights
- Month-to-month customers have highest churn
- Fiber optic users show higher churn rate
- Customers with short tenure churn more
- Electronic check users have higher churn
